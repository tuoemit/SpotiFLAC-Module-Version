# Contributing to SpotiFLAC

Thanks for your interest in contributing! Follow these quick steps:

- Fork the repository and create a feature branch.
- Run tests locally: `python -m pip install -r requirements-dev.txt` then `pytest`.
- Use `ruff` and `black` to format and lint your code.
- Add tests for new behavior and make small, focused PRs.
- For large refactors, open an issue first to discuss the approach.

Be respectful and include a clear description and changelog entry for each PR.
